# Project Overview

## Problem
Students need a simple way to buy and sell used items within their campus community.

## Users
- Buyers
- Sellers
- Admins (future)

## Core flows
1. Browse listings
2. Filter/search listings
3. View details
4. Save favorites
5. Create and manage listings

## Initial scope
- Public listing discovery
- Listing details
- Favorites
- Health/status endpoint
